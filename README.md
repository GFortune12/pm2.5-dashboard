作品名称：绿源·城市呼吸——全国主要城市多污染物大数据交互式分析看板
作品编号2026045218
赛道：大数据应用实践赛
学校：青岛城市学院 作者：王佳，杨瑞琪 指导教师：王超杰
【文件夹说明】本文件夹包含作品的全部源代码、数据集、部署配置文件和提交材料。
【文件清单与说明】
1. app.py
   主程序入口，包含主页（现状总览）的全部功能：
   交互式地图（Folium），气泡颜色随PM2.5浓度分级
   KPI指标卡片（全国年均值、最佳/最差城市）
   右侧城市信息栏（下拉选择城市，展示浓度、排名、污染等级、历年数据）
   侧边栏控制面板（年份切换、污染物切换）
2. pages/1_深度分析.py
   深度分析页面，包含：
   城市排名柱状图（污染最重/最轻Top10）
   城市综合诊断（雷达图、排名趋势、诊断卡片）
   区域污染总结（华北/华南/西部对比分析）
   城市排名波动散点图
   污染物相关性热力图
3. pages/2_趋势分析.py
   趋势分析页面，包含：
   年度趋势预测（线性回归，预测至2026年，展示模型依据）
   治理历程关键事件卡片
   四季卡片点击解读（成因分析 + 健康提示 + 区域对比）
   城市对比与差异解读
   健康与出行建议
   知识小贴士
4. airdata.csv
   原始空气质量数据集（2013-2022年，全国328个城市，约97万条记录）
   包含字段：城市、日期、质量等级、AQI指数、PM2.5、PM10、So2、No2、Co、O3等
5. city_coords.csv
   主要城市经纬度数据，用于Folium地图绘制城市气泡标记
6. style.py
   全局自定义CSS样式文件
7. .streamlit/config.toml
   Streamlit主题配置文件，定义全局配色方案
8. requirements.txt
   Python依赖库清单，可通过一键安装复现运行环境
9. 演示视频.mp4
   作品功能演示视频
10. 作品报告.pdf
    详细的项目报告，包含选题背景、技术方案、核心功能、分析结论等
11. 现场演示PPT.pptx
    演示文稿，包含创意来源、应用对比、技术方案、技术增量、作品展示等

【运行环境与步骤】
1. 安装Python 3.8及以上版本
2. 在终端进入本文件夹，执行：
   pip install -r requirements.txt
3. 启动应用：
   streamlit run app.py
4. 浏览器访问 http://localhost:8501 查看作品
